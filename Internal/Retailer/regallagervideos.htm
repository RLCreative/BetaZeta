How to Link to Regal Lager Videos


Notes:
Invodo links—including the ones here and those in your own web pages—may not work correctly when viewed locally in Internet Explorer.
When viewed in other web browsers (e.g., FireFox), the links will work properly.
When you post your changes to a live web server, the links will behave correctly in all browsers.
When implemented correctly, a new [correctly-sized] window should pop up and display the requested video pod without any "active content" or security warnings.

Adding Video to the Product Pages

Step 1: For compatibility with secure pages, be sure to add the necessary Invodo script references to every page containing Invodo video links (whether in a Video Library list or on a product description page):

<script type="text/javascript" charset="utf-8"> /* <![CDATA[ */if(typeof(Invodo)!='function')document.write('<sc'+'ript src="http'+(document.location.protocol=='https:'?'s':'')+'://e.invodo.com/2.1/js/invodo-auto.js"></sc'+'ript>')/* ]]> */ </script>

For best performance, we recommend adding that line just before the closing </Body> tag on the page, but it can also go in the header. Depending on how your site is set up (with product databases, page templates, etc.), there may be a single file responsible for its insertion on every page, or it may be easier to add the <script> reference in the body of each product description page alongside the links. Regardless, this reference must appear somewhere on the page in order to enable the necessary JavaScript functionality.

Step 2: Design a conspicuous and effective link on each product page to launch Invodo videos for that item. We recommend using a text link that utilizes the product name with the link code provided by Invodo.

To attract a shopper's attention to the link, you can add one of these video icons by copying the appropriate image tag:
